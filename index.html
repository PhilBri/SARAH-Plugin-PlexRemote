Documentation Plugin PlexRemote pour SARAH


auteur : Phi Bri
date : 18/05/2014
version : 0.9


1. CONFIGURATION (PlexRemote.prop)

Le fichier PlexRemote.prop permet de paramétrer les informations nécessaires au bon fonctionnement du plugin.
(Il reste toutefois préférable de renseigner les champs directement dans la console de SARAH.)

Il est donc indispensable de renseigner les éléments suivants :
  1. Champ "client" :
    2. - Renseigner l'adresse IP du serveur Plex, par exemple : 192.168.0.10

  2. Champ "server"
    3. - Renseigner l'adresse IP du client Plex que vous souhaiter controler, par exemple 192.168.0.30

2. DESCRIPTEUR DES COMMANDES (PlexRemote.xml)

La configuration des requètes passe par le fichier externe : PlexRemote.xml.
Celui ci permet de déclarer les actions à demander à Plex afin de les utiliser ensuite de façon simple dans le reste du plugin, sans avoir à manipuler le code javascript.

(Il est à noter que ces requétes sont au format de l'API/Control HTTP qui n'est plus documentée par Plex...)
